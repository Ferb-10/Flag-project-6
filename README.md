# Flag-project-6

## Overview
This project recreates the national flag of Laos using only HTML and CSS.
It focuses on layout creation with CSS positioning, shapes, and color layering without using images.
The goal of this project is to strengthen understanding of CSS positioning and box modeling.

![image alt](https://github.com/Ferb-10/Flag-project-6/blob/b6069c546780caea8e32757cbdc7004e54c3e772/flag%20project.png)

## Features
- Pure HTML and CSS implementation
- Uses position: relative and position: absolute
- Creates geometric shapes using CSS
- Accurate color representation of the Laos flag

## Technologies Used
- HTML5
- CSS

## What I Learned
- How relative and absolute positioning work together
- Layering elements using position and z-axis logic
- Creating circles using border-radius
- Structuring nested div elements for complex layouts
- Using CSS to recreate real-world designs

## 概要
このプロジェクトは、HTMLとCSSのみを使ってラオス国旗を再現した作品です。
画像を使用せず、CSSのレイアウト・配置・図形表現によって構成しています。
CSSのポジショニングやボックスモデルの理解を深めることを目的としています。

## 機能
- HTMLとCSSのみで構成
- position: relative と absolute を使用
- CSSによる図形（円）の作成
- ラオス国旗の配色を再現

## 使用技術
- HTML5
- CSS

## 学んだこと
- relative と absolute の関係性
- CSSによる要素の重なり構造
- border-radius を使った円形の作り方
- ネスト構造を使ったレイアウト設計
- 実物デザインをCSSで再現する考え方
